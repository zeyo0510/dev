May 13, 2020

Expressions are getting smarter

API Tester now has improved support for math and date features in expressions.

Three new methods have been added:

  • math: gives you access to the four basic operations
  • addTime: adds any amount of time to a date
  • formatDate: converts a date from a format to another

The supported date formats are TIMESTAMP and ISO_8601 and you can use all types of assertions on their results.

More information on usage and use-cases in the documentation.
