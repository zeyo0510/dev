SplitUp is a tab manager for Chrome & Firefox, it comes with bunch of features:  
✅ Save sessions \
✅ Split tabs into a new window \
✅ Dark mode \
✅ Drag and Drop tabs across windows \
✅ Fuzzy Search \
✅ Multiscreen support \
✅ Export tabs \
✅ Close windows/tabs \
✅ Mute/Unmute tabs
