Configuration requise .Net Framework 4.52+ (Windows Vista+/Server 2008+). Si vous ne l'avez pas installé, télécharger ici https://www.microsoft.com/fr-FR/download/details.aspx?id=42642

Aucune action requise en plus des exigences système pour l'installation du programme. Vous pouvez exécuter un programme à partir de n'importe quel catalogue, Mais le programme utilisera des paramètres et une base de données individuels, stockés dans le profil utilisateur Windows, en mode normal et dans le catalogue de programmes - en mode portable. Pour démarrer automatiquement le programme lors de la connexion à l'utilisateur, vous devez activer la case appropriée dans la fenêtre Paramètres.

ClipAngelPortable.bat Ou paramètre /p - Exécute l'application en mode portable, Les paramètres et la base de données standard seront chargés / enregistrés dans le même catalogue, où le programme est démarré.

Paramètre /m - Exécute l'application en mode minimisé.
