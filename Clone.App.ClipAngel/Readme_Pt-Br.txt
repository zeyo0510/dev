Para a execução do programa é requerido .Net Framework 4.52+ (Windows Vista + / Server 2008+). Se não estiver instalado, faça o download aqui https://www.microsoft.com/en-us/download/details.aspx?id=42642

Nenhuma outra ação é requerida, além dos requisitos do sistema para instalação do programa. Você pode executar o programa a partir de qualquer catálogo, mas o programa usará configurações e banco de dados individuais, armazenados no perfil do usuário do Windows, no modo normal e no catálogo de programas - no modo portátil. Para auto-iniciar o programa no logon do usuário, você deve ativar a caixa de seleção apropriada na janela Configurações.

ClipAngelPortable.bat ou parâmetro / p - executado o programa no modo portátil, configurações e banco de dados padrão serão carregados / salvos no mesmo catálogo, onde o programa é iniciado.

O parâmetro / m - executa o aplicativo no modo minimizado.
