Program wymaga .Net Framework 4.52+ (Windows Vista+/Server 2008+). Jeśli nie jest zainstalowany, pobierz stąd https://www.microsoft.com/en-us/download/details.aspx?id=42642

Żadne działania nie są wymagane poza wymaganiami systemu do zainstalowania programu. Program można uruchomić z dowolnego katalogu, ale program użyje pojedyńczych ustawień i bazy danych, przechowywanych w profilu użytkownika Windows w normalnym trybie i w katalogu programu - w trybie przenośnym. Aby program uruchamiał się automatycznie przy logowaniu użytkownika, należy włączyć odpowiednie pole wyboru w oknie Ustawień.

ClipAngelPortable.bat lub parametr /p - uruchamia program w trybie przenośnym, ustawienia i standartowa baza danych będzie załadowana/zapisana w tym samym katalogu z którego startuje program.

Parameter /m - uruchamia program w trybie zminilizowanym.
