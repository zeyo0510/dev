Erfordert.Net Framework 4.52+ (Windows Vista+/Server 2008+). Wenn es nicht installiert ist, laden Sie es bitte hier herunter https://www.microsoft.com/en-us/download/details.aspx?id=42642

Es sind keine zusätzlichen Maßnahmen erforderlich, die über die Systemanforderungen für die Installation des Programms hinausgehen. Sie können das Programm aus jedem Verzeichnis ausführen, aber das Programm verwendet einzelne Einstellungen und Datenbanken, die im Windows-Benutzerprofil, im Normalmodus und im Programmverzeichnis - im portablen Modus - gespeichert sind. Um das Programm bei der Benutzeranmeldung automatisch zu starten, müssen Sie das entsprechende Kontrollkästchen im Fenster Einstellungen aktivieren.

ClipAngelPortable.bat oder Parameter /p - führt die Anwendung im portablen Modus aus, Einstellungen und Standarddatenbank werden im selben Verzeichnis geladen/gespeichert, in dem das Programm gestartet wird. Parameter /m - führt die Anwendung im minimierten Modus aus.

ALT+V ist der globale Standard Hotkey zum Öffnen von ClipAngel. Sie können es ändern und andere Hotkeys im Einstellungsfenster finden.

Die Anwendung ändert keine Windows-Dateien oder Registry-Einstellungen (außer, Sie haben die Option Autostart aktiviert).
Die Anwendung hat keine digitale Signatur. Wenn Sie also Smart Screen aktiviert haben, müssen Sie zulassen, dass die Anwendung ausgeführt wird.
