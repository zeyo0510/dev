Das Editcontrol

Mit dem Editcontrol interreagieren Sie mit den Bildern. Rechts ist die Liste und links der
Bearbeitungsbereich. Alles wird im Folgenden Abschnitt genau erläutert.

Die Liste

Über die Liste können ein Bild auswählen, neue Bilder Hinzufügen, Entfernen, die Reihenfolge
ändern und Bilder importieren.

Zum Hinzufügen klicken Sie auf das "", dann wird ein 32x32 Bild eingefügt, oder Sie klicken auf
Pfeil rechts daneben, so dass ein Kontextmenü erscheint, in dem Sie die gewünschte Art von
Bild auswählen können. Sie können die Größe auch mit dem Effekt "Größe ändern" verändern.
Bei "Bild öffnen" im Menü können Sie eine beliebige Bilddatei (bmp,gif,jpg,tif,png,ico...)
importieren. Die Größe des hinzugefügten Bildes wird angepasst, sofern die Quelle nicht größer
als 256x256 ist, ansonsten wird skaliert.

Zum Entfernen klicken Sie auf das "-" in der Leiste. Wenn Sie dann noch die Nachfrage bestätigen,
wird das momentan ausgewählte Bild gelöscht. Dieser Vorgang kann nicht rückgängig gemacht werden.

Mit den Pfeilen können Sie das ausgewählte Bild in der Liste nach oben und unten verschieben.

Wenn Sie gerade ein Bild ausgewählt haben, können Sie die Bildlaufleiste auch mit dem Mausrad
bewegen.
