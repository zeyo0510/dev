Die StandartSymbolleiste

Mit der Standartsymbolleiste haben Sie Zugriff auf die grundlegenden Funktionen
von VariIcons.

Neues Symbol

Hiermit wird die Liste der Bilder im Icon gelöscht und durch drei
leere Bilder mit den Abmessungen 16x16, 32x32 und 48x48 ersetzt.

Symbol öffnen

Hiermit können Sie eine gespeicherte Icondatei einlesen. Es werden
aber nur die 32-bit Bilder geladen. (Hotkey: Strg)

Symbol/Bild speichern

Wenn Sie auf das Icon klicken, wird das aktuell im Editor geladene Symbol
gespeichert. Mit dem KontextMenüpfeil können Sie aber auch das aktuell geladene
Bild als BMP,JPG,GIF,PNG,TIF usw. speichern.
(Hotkeys: Icon speichern: Strg; Bild speichern: Strg???)

Ausschneiden

Hiermit können Sie das aktuelle Bild oder den Auswahlbereich in die Zwischenablage
verschieben. (Hotkey: Strg)

Kopieren

Hiermit können Sie das aktuelle Bild oder den Auswahlbereich in die Zwischenablage
kopieren. (Hotkey: Strg)

Einfügen

Hiermit fügen Sie ein Bild aus der Zwischenablage ein, sofern vorhanden.
Es wird automatisch das Markierwerkzeug ausgewählt.
(Hotkey: Strg)

Rückgängig

Geht einen Bearbeitungsschritt zurück, sofern möglich. Zur Zeit werden
10 Rückgängigoperationen unterstützt.

Wiederholen

Geht einenBearbeitungsschritt nach vorne, sofern möglich. Zur Zeit werden
10 Rückgängigoperationen unterstützt.

AntiAlias

Schaltet Antialiasing ein oder aus. Antialias ist eine Technologie zum Vermindern von
Treppenbildung bei der Rasterung von Vektorgrafiken. Antialias wird für viele
Werkzeuge verwendet.

Compositing Mode

Schaltet die Transparenzüberblendung ein oder aus. Wenn sie eingeschaltet ist,
werden übermalende Farben gemäß ihrer Alpha-Komponente mit der vorhandenen
überblendet, ansonsten ersetzen sie die alte Farbe.

Zentrieren

Schaltet Zentrierung des Bildes im Editcontrol ein oder aus.

Gitter

Zeigt das Raster im Editcontrol an oder verbirgt es.

Einzoomen

Erhöht den Zoom um 1, wenn 'angepasst einzoomen' abgeschaltet ist.

Auszoomen

Erniedrigt den Zoom um 1, wenn 'angepasst einzoomen' abgeschaltet ist.

Angepasst Einzoomen

Stellt den Zoom so ein, dass das Bild im Editcontrol die maximale Größe hat,
bei der es noch ganz zu sehen ist.

Verlauf anzeigen

Zeigt das Farbverlaufsfeld an, sofern das Haupfenster groß genug ist.
