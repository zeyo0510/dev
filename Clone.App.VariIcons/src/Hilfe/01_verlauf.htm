Der Farbverlauf

Mit dem Farbverlauf werden statt der Vordergrundfarbe Verlaufspinsel eingesetzt, so lassen sich leicht kugelige Objekte oder ein Aqua-Effekt erreichen. Verläufe werden angewendet auf "Ellipse", "Gefüllte Ellipse", "Recteck",
"Gefülltes Rechteck", "Kurve", "Gerade Linie", "Stift", "Text", "Polygon", "Gefülltes Polygon", "Airbrush".

Das Verlaufsfenster wird nicht angezeigt, wenn der mit '>' gekennzeichnete Button links daneben nich gedrückt ist und/oder das Fenster zu klein ist, um den Verlauf anzuzeigen, vergrößern Sie in diesem Fall das Fenster.

Zuerst müssen Sie die Verlaufpinsel aktivieren, so dass der Button "Aktivieren" gedrückt ist.

Es stehen 3 verschiedene Pinsel zur Verfügung, die im ersten Listenfeld von rechts angewählt werden:

Radial
   
    >>Der Pinsel füllt eine Form mit einem Strahlfarbverlauf (Anmerkung: leichtes Ruckeln beim Anpassen möglich)
Linear
   
    >>Der Pinsel füllt eine Form mit einem Linearen Farbverlauf
Kreis
   
    >>Der Pinsel füllt eine Form mit einem Farbverlauf in Form eines Kreises

In dem Auswahlfeld darunter können Sie die Farben für den Verlauf einstellen.


Doppelklick fügt eine neue Farbe hinzu, und Rechtsklick auf einen Balken entfernt sie. Wenn ein Balken ausgewählt
ist, erscheint die Begrenzung Rot. Sie können aber nicht weniger als 2 und nicht mehr als 10 Farben in einen Verlauf
einfügen.
Um eine Farbe im Verlauf zu verschieben, klicken Sie sie mit links an, so dass die Begrenzung rot wird und ziehen Sie sie an die gewünschte Stelle.
Wenn Sie eine Farbe ausgewählt haben, können Sie mit dem Farb-Button die Farbe und mit dem Regler darunter die Transparenz regeln.
Mit dem Button "R" wie 'Reverse' können Sie die Reihenfolge der Farben stürzen. Danach ist aber keine Begrenzung
ausgewählt, Sie müssen wieder darauf klicken, um sie auszuwählen.

Im Vorschaufeld können Sie den verlauf drehen und verschieben. Klick mit links und Ziehen verschiebt den Verlauf und Klicken mit rechts und Ziehen in vertikaler Richtung dreht den Verlauf an seinem Nullpunkt.


Im zweiten Listfeld von rechts wählen Sie den Skalierungsmodus aus:

Bereich füllen
     >>Der Verlauf wird auf den gezeichneten Bereich angepasst, so dass z.B. ein Rechteck optimal gefüllt ist.

Bild füllen
    >>Der Verlauf wird auf das gesamte Bild oder den aktuellen Ausschneidebereich skaliert.
